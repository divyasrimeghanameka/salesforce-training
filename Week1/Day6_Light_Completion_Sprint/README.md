# Mini Tasks

## 1. Platform Event Understanding

### Example: College Admission System
When a new student admission happens:

- Administration team receives an alert
- Faculty members are notified
- Payment process begins automatically
- Confirmation email is generated

👉 A single platform event can trigger several automated actions simultaneously.

---

## 2. CLI Reflection

Developers commonly use CLI because:

- It saves development time
- Helps automate repetitive activities
- Useful for handling large Salesforce projects
- Reduces dependency on manual UI operations

---

## 3. Search System Reflection

Efficient search systems are important because:

- Enterprise applications manage massive datasets
- Users require quick record access
- Faster search improves productivity
- Helps in better business decisions
